O que é Hardware?

Hardware é a parte física do computador, composta pelos componentes
eletrônicos, peças e dispositivos que permitem o funcionamento do sistema.

Quais os principais componentes de hardware?

Os hardwares são essenciais para a construção dos computadores. Entre os
principais componentes estão o processador, a memória, a unidade
central de processamento (CPU), o teclado, a placa-mãe, o mouse e a impressora.

O que é memória ROM?

Memória Somente de Leitura (ROM) é um tipo de memória que permite apenas
a leitura dos dados, sem a possibilidade de escrita ou modificação. As informações
são gravadas pelo fabricante uma única vez e não podem ser alteradas ou apagadas,
apenas acessadas. Por isso, a ROM é classificada como memória não volátil.

O que é SETUP?

A palavra "setup" vem do inglês e, em português, pode ser traduzida como
configuração, instalação, organização, disposição ou regulagem. No contexto
da computação, "setup" refere-se a uma configuração que permite ao usuário ajustar
determinados programas para que eles funcionem de acordo com suas necessidades.
Programas como o Google Chrome e sistemas operacionais como o Windows 8
oferecem várias opções de configuração através do setup.

O que é POST?

"Post" é o termo usado para descrever o conteúdo criado e publicado em
uma plataforma online. Esse conteúdo pode estar no formato de imagem,
vídeo, texto, áudio, ou uma combinação desses elementos. As principais
plataformas onde os posts são publicados incluem redes sociais e blogs.
Assim, um post é qualquer conteúdo compartilhado na internet.

O que é BIOS?

O BIOS, que significa “Basic Input and Output System”, é um tipo de
firmware responsável pela inicialização básica da placa-mãe em
equipamentos com plataformas x86, ARM (alguns) e PowerPC. Sua
função é preparar o sistema para que o software subsequente, como
o sistema operacional, possa ser carregado e executado.
